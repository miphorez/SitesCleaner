Sites Cleaner

Расширение разработано в рамках программы "За безопасный интернет для детей". Цель данного расширения в автоматическом скрытии потенциально опасных блоков информации и навигации с популярных сайтов, которыми могут пользоваться дети.

К потенциально опасным зонам для детского интернета на страницах сайтов, по мнению автора расширения, относятся:

  • блоки с рекламой
  • блоки с контентом, отмеченным возрастными ограничениями +16, +18
  • блоки с рекомендуемым контентом по усмотрению авторов сайта
  • блоки с комментариями
  • некоторые навигационные панели, некоторое содержимое меню

Указанные блоки скрываются и становятся недоступными для ребенка при просматривании сайта с включенным расширением. На мультимедийных порталах ребенок не сможет перейти к просмотру неразрешённого контента.
Например, для сайта online-life, доступны будут только фильмы из раздела «избранное». В данной версии расширения фильмы в избранное добавляются родителем при отключенном режиме очистки сайта.

Лицензия

Sites Cleaner - это свободная программа, распространяется на условиях Стандартной общественной лицензии GNU GPLv3. Исходные тексты этой программы доступны на GitHube.

Использованы следующие open source расширения:

  • Stop Autoplay for Youtube™ Extended
  • Vk Comment Blocker
